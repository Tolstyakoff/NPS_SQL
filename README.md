# NPS_SQL
Лабораторная работа по NPS
Оценка NPS менеджеров по работе с клиентами. 
Компания занимается оказанием услуг по разработке дизайна и продвижению личных страниц блогеров. 
В компании работает 12 менеджеров. Каждый менеджер обычно работает c одним клиентом, но допускаются варианты, 
что клиент уходит к другому менеджеру. 
Менеджер работает с клиентом в течении нескольких месяцев, раз в месяц происходит оплата услуг. 
Во время оплаты услуг клиент дает свою оценку удовлетворенности NPS работой менеджера

Задачи:
1. Определить самого эффективного менеджера по:
   - количеству клиентов
   - сумме проектов
   - удержанию клиента (сколько месяцев обслуживался клиент)
   - NPS
2. Найти менеджеров от которых уходят клиенты (Менеджер - количество ушедших клиентов)
3. Оценить клиентов по:
   - доходности (вывести топ 10 клиентов)
   - по стабильности ( не переходил к друшим менеджерам)
   - по длительности (сколько месяцев суммарно обслуживался клиент)
4. Построить динамику клиентов во времени:
   - месяц / среднее кол-во проектов на одного менеджера
   - месяц / NPS по компании
   - месйц / средняя доходность на одного клиента
5. Построить интерактивный дашборд
   элементы управления (выбор менеджера из списка, выбор года)
   общие показатели за все время - NPS, количество клиентов, общая сумма
   график динамика NPS по месяцам
   график динамика выручки (сумма) по месяцам
   график динамика клиентов по месяцам

Сделать
1) с помощью сводных таблиц в Excel
2) c применением  функции QUERY в Google Sheets
